Development environment for testing the Voltage Payment Component library.

Basic Inline Payment

A simple payment component with default settings:

VoltagePayments.create({
  apiKey: "vltg_demo_key",
  walletId: "demo_wallet_id",
  amount: 150000, // 150 sats
  description: "Coffee purchase"
}).mount("#payment-basic");

Customized Payment

Payment component with custom appearance:

VoltagePayments.create({
  apiKey: "vltg_demo_key",
  walletId: "demo_wallet_id",
  amount: 500000, // 500 sats
  description: "Premium service",
  appearance: {
    primaryColor: "#8b5cf6",
    borderRadius: "12px",
    backgroundColor: "#faf5ff"
  }
}).mount("#payment-custom");

Any Amount Payment

Payment component that allows user to specify amount:

VoltagePayments.create({
  apiKey: "vltg_demo_key",
  walletId: "demo_wallet_id",
  amount: null, // User specifies amount
  description: "Donation"
}).mount("#payment-any-amount");
